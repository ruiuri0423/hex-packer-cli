## 1. CRC16 測試 Pattern

### 1.1 基本功能測試

| ID | Pattern 名稱 | 測試輸入 | 預期輸出 | 說明 |
|----|--------------|----------|----------|------|
| CRC-01 | 空資料 | `bytearray([])` | `0x0052` | 初始值 |
| CRC-02 | 單一字節 | `bytearray([0x00])` | `0x81EF` | 基礎計算 |
| CRC-03 | 單一字節 FF | `bytearray([0xFF])` | `0x83ED` | 全 F 輸入 |
| CRC-04 | 連續序列 | `bytearray([0x00, 0x01, 0x02])` | `0x996B` | 多字節計算 |
| CRC-05 | 混合資料 | `bytearray([0xAA, 0xBB, 0xCC, 0xDD])` | - | 模式測試 |

### 1.2 邊界條件測試

| ID | Pattern 名稱 | 測試輸入 | 預期輸出 | 說明 |
|----|--------------|----------|----------|------|
| CRC-06 | 全零輸入 | `bytearray([0x00] * 100)` | - | 長度邊界 |
| CRC-07 | 全 FF 輸入 | `bytearray([0xFF] * 100)` | - | 長度邊界 |
| CRC-08 | 大量資料 | `bytearray([i for i in range(1000)])` | - | 效能測試 |

### 1.3 CRC 計算長度驗證

```
┌─────────────────────────────────────────────────────────┐
│                 CRC 計算長度 Pattern                    │
├─────────────────────────────────────────────────────────┤
│  Pattern A: Buffer[:254] (0x00-0xFD) → CRC MSB@0xFE  │
│  Pattern B: Buffer[256] (完整) → 不同 CRC 值           │
├─────────────────────────────────────────────────────────┤
│  CRC-09: 254 bytes CRC        → 0xFB8D (全零)        │
│  CRC-10: 256 bytes CRC        → 0xD6CF (全零)        │
│  差異驗證: ✓ 不同結果證實長度影響                       │
└─────────────────────────────────────────────────────────┘
```

## 2. Mask Calculator 測試 Pattern

### 2.1 位址對齊測試

| ID | Pattern 名稱 | 輸入 | Base | 預期輸出 | 說明 |
|----|--------------|------|------|----------|------|
| ALIGN-01 | Flash < Base | `0x0500` | `0x1000` | `0x1000` | 對齊到 Base |
| ALIGN-02 | 非 256 對齊 | `0x1234` | `0x1000` | `0x1200` | 對齊到 256 boundary |
| ALIGN-03 | 已是對齊 | `0x1500` | `0x1000` | `0x1500` | 保持不變 |
| ALIGN-04 | 等於 Base | `0x1000` | `0x1000` | `0x1000` | 邊界條件 |
| ALIGN-05 | 大於 Base | `0x2000` | `0x1000` | `0x2000` | 保持不變 |

### 2.2 Mask 位元組計算測試

| ID | Pattern 名稱 | Max Addr | Base | 預期 Bytes | 預期 Bits | 說明 |
|----|--------------|----------|------|-------------|-----------|------|
| MASK-01 | 單一表格 | `0x1000` | `0x1000` | 1 | 1 | 邊界條件 |
| MASK-02 | 兩表格 | `0x1100` | `0x1000` | 1 | 2 | Bit 0, 1 |
| MASK-03 | 多表格 | `0x2000` | `0x1000` | 3 | 17 | Bit 0-16 |
| MASK-04 | 長距離 | `0x5000` | `0x1000` | 9 | 65 | Bit 0, 64 |

## 3. HEX Parser 測試 Pattern

### 3.1 位址解析測試

| ID | Pattern 名稱 | 輸入 | 預期輸出 | 說明 |
|----|--------------|------|----------|------|
| HEX-01 | h 前綴 | `"h00FF"` | `0x00FF` | h 前綴 |
| HEX-02 | 0x 前綴 | `"0xFF"` | `0xFF` | 0x 前綴 |
| HEX-03 | 無前綴 | `"FF"` | `0xFF` | 直接十六進位 |
| HEX-04 | 小寫 | `"hff"` | `0xFF` | 小寫轉換 |
| HEX-05 | 大寫 | `"0xABCD"` | `0xABCD` | 大寫支援 |
| HEX-06 | 含空白 | `"  hFF  "` | `0xFF` | 空白處理 |
| HEX-07 | 整數輸入 | `255` | `255` | 整數直通 |
| HEX-08 | 空字串 | `""` | `-1` | 錯誤處理 |
| HEX-09 | 無效字串 | `"invalid"` | `-1` | 錯誤處理 |
