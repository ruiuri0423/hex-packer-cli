## 🏗️ 模組化架構

### 重構後的目錄結構

```
hex-packer-cli/
├── src/
│   ├── main.py              # 主程式 (引用 core 模組)
│   └── core/                # ⭐ 核心模組套件
│       ├── __init__.py      # 統一的匯出入口
│       ├── crc16.py         # CRC16 平行計算 (硬體對應)
│       ├── hex_parser.py    # HEX 位址解析
│       ├── mask_calculator.py  # Flash 位址對齊 & Mask 計算
│       ├── register.py      # 寄存器 Excel 解析
│       ├── packer.py         # 固件打包器核心
│       └── viewer.py         # 寄存器查看器
├── tests/                   # ⭐ 測試框架
│   ├── test_crc16.py        # CRC16 單元測試
│   ├── test_mask.py         # Mask Calculator 單元測試
│   └── test_hex_parser.py   # HEX Parser 單元測試
├── requirements.txt         # ⭐ 依賴管理
├── pytest.ini               # ⭐ pytest 配置
├── test_samples/            # 測試樣本
└── docs/                    # 文件
```

## 類別架構

### 1. `FirmwarePacker` - 固件打包器
**主要方法：**
- `load_csv_config()` - 載入 CSV 配置
- `save_csv_config()` - 儲存 CSV 配置
- `calculate_addresses()` - 計算動態位址
- `generate_firmware_hex()` - 生成固件 HEX

---

### 2. `RegisterViewer` - 寄存器查看器
**主要方法：**
- `add_gap_to_map()` - 添加 Gap 映射
- `process_and_map_excel()` - 處理 Excel 並映射
- `update_treeview()` - 更新 Treeview

---

### 3. `AppGUI` - GUI 應用程式
**主要分頁：**
- Tab 1: Firmware Packer
- Tab 2: M0 HEX Converter
- Tab 3: Register Integrator
- Tab 4: Single Map Viewer
- Tab 5: Logic Test
