## 核心演算法

### CRC16 計算
```
多項式：0x8005
初始值：0x0052
位元運算：16-stage 並行邏輯
```

### Enable Mask 計算
```
Bit Position = (Flash_Address - Base_Address) // 256
Mask = Σ (1 << bit_position) for all enabled tables
```

## 數據流

```
輸入來源：
  ├── CSV Config (.csv) → FirmwarePacker
  ├── Excel Registers (.xlsx) → RegisterViewer
  ├── Intel HEX (.hex) → M0 Converter → FirmwarePacker
  └── Raw HEX (.txt) → M0 Converter → FirmwarePacker

處理管道：
  CRC16 計算 → 參數注入 → Mask 計算 → 緩衝填充

輸出結果：
  └── Firmware HEX (.hex) - 最終固件檔案
```
